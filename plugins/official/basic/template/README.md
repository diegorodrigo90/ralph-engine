# Ralph Engine Basic Starter

This starter gives a minimal runtime-owned workspace contract for teams that want a small, reviewable implementation loop.

## Included files

- `config.yaml` — baseline runtime configuration for one focused work item at a time
- `hooks.yaml` — lightweight hook contract with required test execution
- `prompt.md` — starter prompt guidance for project context and runtime rules

## Next steps

1. Replace the placeholder project context in `prompt.md`.
2. Fill the hook commands that your repository actually requires.
3. Adjust agent and workflow settings in `config.yaml` to match your team pace.
