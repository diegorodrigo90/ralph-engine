O Ralph Engine é instalado a partir do código-fonte. A distribuição por npm e Homebrew está preparada, mas intencionalmente desabilitada até o pipeline de release estar totalmente validado.

## Pré-requisitos

- Git
- Rust 1.91.1 (fixado via `rust-toolchain.toml`)
- Node.js 20.19.0 (fixado via `.tool-versions`)

## Instalação pelo Código-Fonte

Clone o repositório:

```bash
git clone https://github.com/diegorodrigo90/ralph-engine.git
```

Execute o script de bootstrap:

```bash
cd ralph-engine && ./scripts/bootstrap-dev.sh
```

Confirme que o binário da CLI compila e executa:

```bash
cargo run -p re-cli -- --version
```

### O que o bootstrap-dev.sh faz

O script de bootstrap prepara o ambiente local de desenvolvimento em um único passo:

1. **Sincronização de toolchain** — executa `asdf install` se o asdf estiver disponível com os plugins de rust e nodejs, garantindo que você tenha exatamente as versões fixadas em `.tool-versions` e `rust-toolchain.toml`.
2. **Dependências Node** — executa `npm ci` na raiz e dentro do diretório `site/`.
3. **Ferramentas de desenvolvimento** — executa `scripts/install-dev-tools.sh` para instalar as ferramentas Cargo necessárias (cargo-llvm-cov, cargo-deny, cargo-audit, etc.).
4. **Git hooks** — executa `lefthook install` para configurar hooks de pre-commit e pre-push.

Se o asdf não estiver instalado, o script continua com qualquer toolchain que já esteja no seu PATH.

## Verificar a Instalação

Exiba a ajuda da CLI:

```bash
cargo run -p re-cli -- --help
```

Exiba a ajuda em português:

```bash
cargo run -p re-cli -- --locale pt-br --help
```

Execute o contrato completo de validação local:

```bash
./scripts/validate.sh --mode local
```

Execute a suíte completa de testes:

```bash
cargo test --workspace --all-targets
```
