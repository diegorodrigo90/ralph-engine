## Instalar

<Tabs syncKey="pkg">
  <TabItem label="npm">
    Instalar globalmente:

    ```bash
    npm install -g ralph-engine
    ```

    Ou executar sem instalar:

    ```bash
    npx ralph-engine --help
    ```

    Também funciona com pnpm e yarn:

    ```bash
    pnpm add -g ralph-engine
    ```

    ```bash
    yarn global add ralph-engine
    ```
  </TabItem>
  <TabItem label="Homebrew">
    ```bash
    brew install ralph-engine
    ```
  </TabItem>
  <TabItem label="Cargo">
    Compila a partir do código-fonte (requer Rust):

    ```bash
    cargo install ralph-engine
    ```
  </TabItem>
</Tabs>

:::note
Todos os 8 plugins oficiais já vêm incluídos. Nenhuma instalação adicional necessária.
:::

## Verificar

Confirme que a CLI está disponível:

```bash
ralph-engine --version
```

Exiba os comandos disponíveis:

```bash
ralph-engine --help
```

Mude para português:

```bash
ralph-engine --locale pt-br --help
```

Execute uma verificação de saúde no seu projeto:

```bash
ralph-engine doctor
```

## Compilar a partir do Código-Fonte

Se preferir compilar o binário manualmente:

```bash
git clone https://github.com/diegorodrigo90/ralph-engine.git
```

```bash
cd ralph-engine && ./scripts/bootstrap-dev.sh
```

```bash
cargo build --workspace --release
```

O binário estará em `target/release/ralph-engine`.
