## 1. Instalar

<Tabs syncKey="pkg">
  <TabItem label="npm">
    ```bash
    npm install -g ralph-engine
    ```
  </TabItem>
  <TabItem label="Homebrew">
    ```bash
    brew install ralph-engine
    ```
  </TabItem>
  <TabItem label="Cargo">
    ```bash
    cargo install ralph-engine
    ```
  </TabItem>
</Tabs>

## 2. Inicializar um projeto

Crie a configuração inicial do projeto:

```bash
ralph-engine templates materialize official.basic.starter ./meu-projeto
```

Isso cria um diretório `.ralph-engine/` com:
- `config.yaml` — configuração do projeto
- `prompt.md` — conteúdo de prompt específico do projeto

## 3. Verificar o setup

```bash
ralph-engine doctor
```

Se tudo estiver verde, está pronto.

## Próximos passos

- Execute `ralph-engine --help` para explorar todos os comandos
