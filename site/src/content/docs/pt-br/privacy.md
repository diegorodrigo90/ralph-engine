## O que coletamos

Nada. Este site não coleta dados pessoais, não define cookies, não usa analytics e não rastreia visitantes.

## Busca

A busca do site é feita pelo Pagefind, que roda inteiramente no seu navegador. Nenhuma consulta de busca é enviada para qualquer servidor.

## Hospedagem

Este site é hospedado no Cloudflare Pages. O Cloudflare pode processar endereços IP em logs de servidor como parte da operação padrão de CDN. Veja a [Política de Privacidade do Cloudflare](https://www.cloudflare.com/privacypolicy/) para detalhes.

O Cloudflare Pages não define cookies neste site. Gerenciamento de bots e desafios WAF não estão habilitados.

O Ralph Engine é software de código aberto licenciado sob MIT. O código-fonte deste site está disponível no [repositório do Ralph Engine](https://github.com/diegorodrigo90/ralph-engine).
