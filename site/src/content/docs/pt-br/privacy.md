**Última atualização:** Abril de 2026

O Ralph Engine é um projeto de código aberto. Esta política descreve como tratamos dados neste site.

## O que coletamos

Este site não coleta dados pessoais diretamente. Não utilizamos formulários, contas de usuário, sistemas de login ou formulários de contato.

## Analytics

Utilizamos o **Cloudflare Web Analytics** para entender como o site é utilizado. O Cloudflare Web Analytics é uma ferramenta de analytics que prioriza a privacidade e que:

- **Não** utiliza cookies
- **Não** rastreia usuários por endereço IP
- **Não** faz fingerprinting de visitantes
- **Não** utiliza localStorage ou qualquer estado no lado do cliente

Ele coleta apenas métricas anônimas e agregadas: visualizações de página, referências, tipo de navegador, país e dados de desempenho. Nenhum dado pessoal é coletado ou armazenado.

## Busca

A busca do site é feita pelo **Pagefind**, que roda inteiramente no seu navegador. Nenhuma consulta de busca é enviada para qualquer servidor.

## Hospedagem

Este site é hospedado no **Cloudflare Pages**. Como parte da operação padrão de CDN, o Cloudflare processa endereços IP de visitantes para entrega de conteúdo e segurança. Este processamento é realizado pelo Cloudflare como operador de dados. Veja a [Política de Privacidade do Cloudflare](https://www.cloudflare.com/privacypolicy/) para detalhes.

## Cookies

Este site não define nenhum cookie.

## Código aberto

O Ralph Engine é software de código aberto licenciado sob a Licença MIT. O código-fonte deste site está disponível no [repositório do Ralph Engine](https://github.com/diegorodrigo90/ralph-engine).

## Seus direitos

Sob as leis de proteção de dados aplicáveis (incluindo a LGPD e o GDPR), você tem o direito de acessar, corrigir ou excluir quaisquer dados pessoais que mantenhamos sobre você. Como não coletamos dados pessoais, não há dados para acessar, corrigir ou excluir.
