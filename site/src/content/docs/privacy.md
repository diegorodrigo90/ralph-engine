**Last updated:** April 2026

Ralph Engine is an open-source project. This policy describes how we handle data on this website.

## What we collect

This website does not collect personal data directly. We do not use forms, user accounts, login systems, or contact forms.

## Analytics

We use **Cloudflare Web Analytics** to understand how the site is used. Cloudflare Web Analytics is a privacy-first analytics tool that:

- Does **not** use cookies
- Does **not** track users via IP address
- Does **not** fingerprint visitors
- Does **not** use localStorage or any client-side state

It collects anonymous, aggregate metrics only: page views, referrers, browser type, country, and performance data. No personal data is collected or stored.

## Search

Site search is powered by **Pagefind**, which runs entirely in your browser. No search queries are sent to any server.

## Hosting

This site is hosted on **Cloudflare Pages**. As part of standard CDN operation, Cloudflare processes visitor IP addresses for content delivery and security. This processing is performed by Cloudflare as a data processor. See [Cloudflare's Privacy Policy](https://www.cloudflare.com/privacypolicy/) for details.

## Cookies

This website does not set any cookies.

## Open source

Ralph Engine is open-source software licensed under the MIT License. The source code for this website is available in the [Ralph Engine repository](https://github.com/diegorodrigo90/ralph-engine).

## Your rights

Under applicable data protection laws (including the LGPD and GDPR), you have the right to access, correct, or delete any personal data we hold about you. Since we do not collect personal data, there is no data to access, correct, or delete.
