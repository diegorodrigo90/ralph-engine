## Install

<Tabs syncKey="pkg">
  <TabItem label="npm">
    Install globally:

    ```bash
    npm install -g ralph-engine
    ```

    Or run without installing:

    ```bash
    npx ralph-engine --help
    ```

    Also works with pnpm and yarn:

    ```bash
    pnpm add -g ralph-engine
    ```

    ```bash
    yarn global add ralph-engine
    ```
  </TabItem>
  <TabItem label="Homebrew">
    ```bash
    brew install ralph-engine
    ```
  </TabItem>
  <TabItem label="Cargo">
    Compiles from source (requires Rust):

    ```bash
    cargo install ralph-engine
    ```
  </TabItem>
</Tabs>

:::note
All 8 official plugins are bundled. No separate install needed.
:::

## Verify

Check that the CLI is available:

```bash
ralph-engine --version
```

Show available commands:

```bash
ralph-engine --help
```

Switch to Portuguese:

```bash
ralph-engine --locale pt-br --help
```

Run a health check on your project:

```bash
ralph-engine doctor
```

## Build from Source

If you prefer to build the binary yourself:

```bash
git clone https://github.com/diegorodrigo90/ralph-engine.git
```

```bash
cd ralph-engine && ./scripts/bootstrap-dev.sh
```

```bash
cargo build --workspace --release
```

The binary will be at `target/release/ralph-engine`.
