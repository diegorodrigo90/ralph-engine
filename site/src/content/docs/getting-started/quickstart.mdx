## 1. Install

<Tabs syncKey="pkg">
  <TabItem label="npm">
    ```bash
    npm install -g ralph-engine
    ```
  </TabItem>
  <TabItem label="Homebrew">
    ```bash
    brew install ralph-engine
    ```
  </TabItem>
  <TabItem label="Cargo">
    ```bash
    cargo install ralph-engine
    ```
  </TabItem>
</Tabs>

## 2. Initialize a project

Scaffold a starter project configuration:

```bash
ralph-engine templates materialize official.basic.starter ./my-project
```

This creates a `.ralph-engine/` directory with:
- `config.yaml` — project configuration
- `prompt.md` — project-specific prompt content

## 3. Check your setup

```bash
ralph-engine doctor
```

If everything is green, you're ready to go.

## What to do next

- Run `ralph-engine plugins list` to see available plugins
- Run `ralph-engine --help` to explore all commands
- Read [Configuration](/guides/configuration/) to customize your project
- Read [CLI Commands](/reference/cli/) for the full command reference
