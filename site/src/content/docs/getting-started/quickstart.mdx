## Bootstrap

Install the pinned toolchain, Node dependencies, dev tools, and git hooks:

```bash
./scripts/bootstrap-dev.sh
```

## Validate

Prove the project is healthy before you start changing it:

```bash
./scripts/validate.sh --mode local
```

## Run

Confirm the Rust-first CLI baseline is working:

```bash
cargo run -p re-cli
```

## After the first run

The next useful paths depend on what you want to do:

- Read [Architecture](/reference/architecture) for the system model.
- Read [Extending](/guides/plugins) for the extension surface.
- Read [Coding Standards](/development/coding-standards) to contribute code.
- Read [CLI Reference](/reference/cli) to explore available commands.
